AI Audience Agent

Overview

AI Audience Agent는 VR 발표 환경에서 발표자의 발화를 인식하고, ASR–LLM–TTS 파이프라인을 통해 실시간으로 질문과 반응을 생성하는 가상 청중 시스템입니다. 사용자는 VR 헤드셋을 착용한 상태에서 발표를 진행하며, 에이전트는 적절한 타이밍에 시선·표정·제스처와 함께 질문을 던집니다.

본 프로젝트에서 저는 전체 상호작용 흐름과 기술 구조를 설계하고, 음성 인식, 언어 모델, 음성 합성 모듈을 Unity 기반 VR 환경과 연동하는 역할을 담당했습니다.

System Architecture

시스템은 다음과 같은 실시간 파이프라인으로 구성됩니다.

  • 1. ASR (Speech-to-Text) — 발표자의 음성을 스트리밍으로 인식하여 텍스트로 변환
  • 2. LLM (Question Generation) — 인식된 텍스트와 발표 흐름을 바탕으로 후속 질문 및 백채널 문장 생성
  • 3. TTS (Text-to-Speech) — 생성된 문장을 에이전트의 음성으로 합성
  • 4. VR Agent Control — 음성 타이밍에 맞춰 시선, 고개 끄덕임, 표정 변화, 상체 제스처를 제어

특히, ASR–LLM–TTS 과정에서 발생하는 지연(latency)을 고려하여 대기 시간 동안 에이전트가 ‘생각하고 있는 듯한’ 몸짓을 취하도록 설계함으로써, 시스템 지연을 사회적 신호로 전환하는 데 집중했습니다.
