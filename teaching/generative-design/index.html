Teaching / Generative Design
한국예술종합학교 미술원 디자인과 예술전문사 과정

Generative Design

규칙·우연·피드백을 중심으로 생성적 사고를 훈련하고, 인공 생명 기반의 제너러티브 에코시스템을 구현하는 스튜디오형 강의.

Overview

제너러티브 디자인 2는 Artificial Life를 주제로, 생명-유사 규칙이 형태와 행동을 어떻게 창발시키는지 탐구하는 스튜디오 수업이다. 학생들은 규칙 설계, 시뮬레이션, 피드백, 리파인을 반복하며 최종적으로 하나의 Generative Ecosystem 작업을 완성한다.

Keywords

Artificial Life, Generative Ecosystem, Simulation, Three.js, Blender Geometry Nodes

Context

한국예술종합학교 미술원 디자인과 예술전문사 과정에서 진행한 수업으로, 생성 규칙과 피드백 루프를 바탕으로 시각적·행동적 시스템을 설계하는 실습 중심 스튜디오 형식으로 운영되었다.

Weekly Topics

Perlin / Simplex Noise, Cellular Automata, Reaction–Diffusion, Boids, L-System, Genetic Algorithm, Slime Mold, Predator–Prey Ecosystem, A/V Mapping, Interactive UI, Performance Tuning.

Outcome

학생들은 세포 → 개체 → 생태계로 확장되는 생성 시스템을 설계하고, 학기 말에는 자신만의 Generative Ecosystem을 온라인 전시 형태로 공개한다.
