AI Audience Agent · System View

System Architecture

AI Audience Agent는 다음과 같은 파이프라인으로 동작합니다.

  • ASR — 발표자의 음성을 실시간 스트리밍으로 인식
  • LLM — 발표 내용과 흐름에 맞는 질문·백채널 생성
  • TTS — 생성된 텍스트를 에이전트 음성으로 변환
  • VR Agent Control — 음성과 함께 시선·고개 끄덕임·표정·상체 제스처 제어

기술적 지연을 숨기기보다, 기다리는 동안의 몸짓을 디자인함으로써 시스템의 한계를 하나의 사회적 신호로 전환했습니다.
